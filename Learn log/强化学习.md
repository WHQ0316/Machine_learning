# 强化学习

强化学习是Agent在与环境的互动当中为了达成目标而进行的学习过程 （试错学习）

---

基本元素：

- Agent （主体）
- Environment （环境）
- Goal （目标）

---

主要元素：

- state （状态）
- Action （行动）
- Reward （奖励）

---

核心元素：

- Police （策略）
- Value （价值）

---

两个特点：

- Trial and Error （试错）
- Delayed Reward (延迟奖励)

---

核心问题：

- Exploration
- Exploitation

---
